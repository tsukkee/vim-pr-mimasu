# 何をするVimプラグインか
このVimプラグインは、GitHubのPull Requestのレビューを効率化することを目的とする

# 動作条件
* サポートするVimのバージョン Vim 9.1以降 (Neovimは一旦考慮しない)
* ghコマンドラインツールがインストールされていること

# 今回実装する機能
* `:Mimasu`コメンドで現在のリポジトリの現在のブランチのPull Requestで、差分があるファイル一覧をツリー表示にする
* ツリー上のファイルを選択すると、左半分に変更前、右半分に現在のファイルを開き、それぞれを差分表示にする
  * 現在のファイルを使うことで、LSPの支援を受けて変更箇所とそこに関連するファイルを簡単に確認できるようにする

# 実装方針
* GitHubの認証がめんどくさいので、認証済みのghコマンドが有効である前提で実装する
* autoloadを活用して必要なときのみコードが読み込まれるようにする
* docにドキュメントを用意する

# 参考情報
* ghコマンドのリファレンスは https://cli.github.com/manual/ から探すこと
  * gh なんとか --help も必要に応じて使ってOK
